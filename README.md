# Lumbung

A trustless rotating savings circle (ROSCA / *arisan*) on Stellar that doubles as an AI-powered credit ladder for unbanked Indonesian SMEs.

Built for the **APAC Stellar Hackathon 2026** — Pillar 1: Local Finance & Real World Access.

## Problem

69.5% of Indonesian SMEs have no access to bank credit, despite contributing 61% of GDP — mainly due to lack of collateral and formal credit history.

## How It Works

- Merchants join small savings circles. Contributions and payouts move through a smart contract, not a human treasurer.
- Completing a cycle without default builds an on-chain reputation.
- An ML model scores that reputation to unlock revenue-based working capital, repaid automatically as a small cut of each sale.
